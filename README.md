# 🌱 Climate Crop AI

AI-powered crop recommendation system that helps farmers choose the best crops based on climate conditions.

## Features

• AI Crop Recommendation
• Climate Risk Score
• Interactive Crop Planner
• Weather Insights Dashboard
• AI Plant Doctor (Leaf Disease Scanner)
• Farming Chatbot Assistant

## Tech Stack

Frontend:
HTML
CSS
Tailwind
JavaScript
Chart.js

Backend:
Python
Flask

## How it Works

1. User enters location and climate data
2. AI analyzes rainfall, temperature, soil and season
3. System ranks crops based on resilience score
4. User receives crop recommendations and insights
